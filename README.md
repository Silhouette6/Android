# Android整理的一些线刷工具



## #Fastboot/ADB

> 基本上是最通用的一种刷机方法,几乎适用所有设备





## #高通QPST（9008）

> 底层，底层，底层，非常底层的一种刷机方式，通过firehose协议对emmc/ufs进行直接的下载操作





## #以下是一些手机厂商的专门刷机工具



### %Mi Flash（小米）

> 小米开发的刷机工具，不仅支持fastboot协议，还支持firehose协议。功能还是很强大的

### %LGUP（LG）

> LG的刷机工具，对LG手机用，且手机要进入Download模式下进行，用来刷固件的

### %Odin（三星）

跟LGup有异曲同工，也是只能用来刷一点固件的东西
